# Roadmap Node Types

Root Node

Purpose:
- Represents entire roadmap

Characteristics:
- Largest node
- Strong glow
- Progress ring

Topic Node

Purpose:
- Major roadmap section

Characteristics:
- Medium node
- Expandable
- Interactive

Pattern Node

Status:
Planned

Purpose:
- Individual learning pattern

Examples:
- Prefix Sum
- Sliding Window
- DFS
- BFS

Characteristics:
- Smaller node
- Secondary hierarchy
