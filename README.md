# Interview Atlas

Interview Atlas is an open-source interview preparation platform built with React, TypeScript, Tailwind CSS, and Supabase.

## ✨ Features

- 📚 DSA Roadmap
- 💻 CS Fundamentals
- 📈 Progress Tracking
- 📝 Personal Notes
- 🔖 Bookmarks
- 🔐 Authentication
- 👥 Role-Based Access (Guest, Member, Admin, Super Admin)

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- PostgreSQL

## 🚀 Status

Currently under active development.
